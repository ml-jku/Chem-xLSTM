# ChEMBL v31 SMILES Dataset

This dataset contains **1.9 million canonical SMILES strings** extracted from ChEMBL v31 by Özçelik et al., designed for molecular modeling and drug discovery tasks.

## Dataset Overview
- **Source:** [ChEMBL v31](https://www.ebi.ac.uk/chembl/)
- **Format:** Canonical SMILES as text
- **Size:** 1.9M molecules

## Data Splits
additionally to Özçelik et al., we split the validation set randomly into 50% validation and 50% test set.


# References
